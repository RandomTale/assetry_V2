# Assetry — Quick Start Guide

This project is fully ready to run! 

## Step 1: Install Dependencies & Run Backend
1. Open a terminal and navigate to the `server` folder:
   cd server
2. Install the backend dependencies:
   npm install
3. Start the backend:
   npm run dev

*(Note: On the first run, the backend will show a Warning about missing Firebase credentials. This is expected until you configure your Firebase project in the `server/.env` file).*

## Step 2: Install Dependencies & Run Frontend
1. Open a new, separate terminal and navigate to the `client` folder:
   cd client
2. Install the frontend dependencies:
   npm install --legacy-peer-deps
3. Start the frontend:
   npm run dev

## Step 3: View the App
Open your web browser and go to: http://localhost:3000

Enjoy using the marketplace!
